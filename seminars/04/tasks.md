## Задания к семинару №4

**Замечание 1**: во всех нижеприведённых заданиях некорректный ввод должен обрабатываться, при этом пользователю должно выводиться сообщение об ошибке.

**Замечание 2**: ввод/вывод любых данных должен сопровождаться "приглашением"/пояснением (к примеру, "Введите радиус окружности: ").

### Массивы

1. Считайте длину массива и массив из целых чисел (элементы массива записаны на одной строке через пробел). Выведите среднее арифметическое всех чисел.
2. Считайте длину массива и массив из целых чисел (элементы массива записаны на одной строке через пробел). Выведите количество чётных и нечётных чисел в нём.
3. Считайте длину массива и массив из целых чисел (элементы массива записаны на одной строке через пробел). Обменяйте местами первый и последний элемент массива. Выведите получившийся массив.
4. Считайте длину массива и массив из целых чисел (элементы массива записаны на одной строке через пробел). Обменяйте местами элементы на чётных и нечётных позициях и выведите результат. То есть, должны попарно поменяться местами элементы с индексами 0 и 1, с индексами 2 и 3 и т.д. Например, для массива `[ 1, 2, 3, 4, 5, 6, 7 ]` ответом будет ` [ 2, 1, 4, 3, 6, 5, 7 ]`.
5. Считайте длину массива и два массива из целых чисел одинакового размера (элементы массива записаны на одной строке через пробел). Попарно перемножьте элементы двух массивов, составив новый массив. Выведите результирующий массив.
6. Считайте длину массива и массив из целых чисел (элементы массива записаны на одной строке через пробел). Составьте новый массив, который состоит только из положительных чисел исходного массива.

### Строки

1. Считайте строку. Разверните исходную строку в обратном порядке и сохраните в новую переменную. Выведите получившуюся строку. **Методы стандартной библиотеки типа `reverse` использовать запрещено.**
2. Считайте строку. Переведите все буквы из нижнего регистра в верхний, а из верхнего регистра - в нижний. Таким образом, для строки `Hello, World!` ответом будет `hELLO, wORLD!`.
3. Считайте две строки. Проверьте, начинается ли первая строка со второй (т.е. вторая строка является префиксом первой).
4. Считайте строку, состоящую из одного или нескольких слов (слова разделены пробелами). В массиве, состоящем из слов, замените на `null` каждое слово, содержащее букву `Q`. После этого выведите длину каждого слова (строки со значением `null` пропустите).
